# Job Application Tracker

A full-stack web application to track job applications, built using **ASP.NET Core Web API** and **React** with **MUI (Material UI)** for styling.

## Features

- Add, edit job applications
- View a list of all applications
- Sort applications by date, position, or status
- Pagination for better navigation
- Modern responsive UI with Material UI
- API error handling and input validation

## Technologies Used

- Frontend: React, MUI (Material UI), Axios, Vite
- Backend: ASP.NET Core Web API, Entity Framework Core
- Database: SQLLite
- Tools: Visual Studio 2022, Visual Studio Code, Git, GitHub

## Getting Started

### Prerequisites

- Node.js and npm
- .NET 6 SDK or later
- Git (for version control)

### Backend Setup

1. Open the solution in Visual Studio 2022
2. Set `JobApplicationTrackerAPI` as the startup project
3. Run the application — Swagger UI should open at `https://localhost:44328/swagger`

### Frontend Setup

1. Navigate to the `client` folder in your terminal
2. Install dependencies:

   ```bash
   npm install
   npm run dev

3. Open http://localhost:5173 to use the app
